# OO-project
Object-Oriented analysis and design Project

## Project running usage

### Complier 

進到某個檔案底下執行
Example : complie /src/MyGraphic/class_shape.java
```shell
javac -d ../../classes -cp ../../classes class_shape.java
```
### Execute

進到src目錄下執行MainFrame
```shell
java -cp ../classes MainFrame
```
